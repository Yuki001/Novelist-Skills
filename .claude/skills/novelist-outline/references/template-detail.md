# 细纲模板

## 场景规划

### 场景 [编号]
- 章节归属：[属于第几章]
- 视点人物：[从谁的视角叙述]
- 场景标题：[场景的简短标题]
- 时间：[场景发生的时间]
- 地点：[场景发生的地点]
- 在场角色：[参与场景的所有角色]

#### 主动场景（目标驱动）
- 目标：[视点人物想要达成什么]
- 冲突：[遇到什么阻碍]
- 挫折：[结果如何，是否达成目标]

#### 被动场景（反应驱动）
- 反应：[对挫折的情绪反应]
- 困境：[面临的两难选择]
- 决定：[最终做出什么决定]

#### 场景要素
- 悬念钩子：[场景结尾的钩子]
- 信息揭示：[揭示了什么新信息]
- 人物变化：[角色有什么变化]
- 推进作用：[如何推动主线发展]
