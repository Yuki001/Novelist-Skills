# 黄金开篇执行模块

前三章或前一万字内严格执行以下规则：

## 1. 动态场景切入

禁止使用静止场景或大段环境描写作为开头。必须从充满冲突、动作或激烈对话的动态事件中直接切入故事。

## 2. 核心冲突前置

第一章内必须抛出主角面临的核心冲突或巨大困境，让读者立刻明白故事的看点所在。

## 3. 避免信息轰炸

绝对禁止在开篇大篇幅介绍世界观、人物背景。所有必要信息应在后续情节推进中，通过角色的行动和对话"滴灌式"透露给读者。

## 4. 限制出场人数

开篇（第一章）有名有姓、参与核心冲突的角色不应超过三人，确保焦点集中在主角身上。

## 5. 快速展现金手指

在主角遇到第一个危机后，应在第二章或第三章内快速展现其"金手指"（核心信息差）的作用，完成第一个小的"逆袭"或"打脸"，给读者带来初步的爽感。
