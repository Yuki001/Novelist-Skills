# 质量检查清单

交付章节前使用此清单确保质量。

## 整体检查

### 基础要素

- [ ] **章节有明确标题**
- [ ] **字数符合预期**
- [ ] **章节完整性**
- [ ] **时间地点清晰**

## 开头检查

- [ ] **前 3 段内抓住读者**
- [ ] **与上一章有连接**
- [ ] **背景信息不过量**

## 内容检查

### 情节推进

- [ ] **本章有核心事件**
- [ ] **推动主线剧情**
- [ ] **逻辑自洽**

### 冲突与张力

- [ ] **有明确冲突**
- [ ] **张力有变化**
- [ ] **有转折或新信息**

## 人物检查

- [ ] **人物行为一致**
- [ ] **人物有反应**
- [ ] **人物有声音**
- [ ] **人物展示而非讲述**

## 对话检查

- [ ] **每句对话有目的**
- [ ] **对话简洁自然**
- [ ] **有潜台词**
- [ ] **标签使用正确**

## 悬念检查

- [ ] **结尾有钩子**
- [ ] **悬念强度适当**
- [ ] **不是虚假悬念**
- [ ] **为下一章铺垫**

## 展示而非讲述检查

- [ ] 是否直接陈述情绪？（改为身体反应）
- [ ] 是否用形容词总结？（改为具体描写）
- [ ] 是否跳过了关键场景？（补充展示）

## 节奏检查

- [ ] **句子长度有变化**
- [ ] **段落长度适当**
- [ ] **信息密度有变化**

## 语言检查

- [ ] **没有 AI 写作痕迹**
- [ ] **"的"字不密集**
- [ ] **用词精确**

## 连贯性检查

- [ ] **与前文连贯**
- [ ] **伏笔有呼应**
- [ ] **时间线一致**

## 交付前最终检查

- [ ] 通读全文，无错别字
- [ ] 标点符号正确
- [ ] 对话标签正确
- [ ] 段落划分清晰
- [ ] 格式一致
- [ ] 如果是续章，确认与前文的连贯性
